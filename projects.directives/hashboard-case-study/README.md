# Zettahash Hashboard Case Study

This folder contains a standalone case study for the Zettahash Hashboard frontend. It explains the project purpose, product strategy, UX decisions, and implementation approach, and includes illustrative screenshots and mockups.

## Contents

- `case-study.md` - full narrative case study
- `purpose.md` - concise purpose statement and non-goals
- `screenshots/` - illustrative UI screenshots (concepts)
- `mockups/` - wireframes and UI mockups

## Notes

- The visuals in `screenshots/` and `mockups/` are intentionally lightweight SVGs so they can live in source control without large binaries.
- Replace these SVGs with real captures or high-fidelity exports when you are ready to publish.
