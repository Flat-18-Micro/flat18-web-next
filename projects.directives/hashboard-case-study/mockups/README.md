# Mockups and Wireframes

These mockups capture layout and information hierarchy decisions. They are intentionally simple so they can be edited quickly.

## Files

- `wireframe-dashboard.svg` - wireframe layout for the overview page
- `wireframe-mining.svg` - wireframe layout for mining metrics
- `mobile-quickview.svg` - mobile quickview layout
- `ui-kit.svg` - compact UI kit showing card, tile, and chart styles
