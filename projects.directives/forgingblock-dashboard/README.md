# rood

This folder contains the project case study and visual assets (screenshots and mockups).

## Contents
- `CASE_STUDY.md` - comprehensive case study for the ForgingBlock dashboard payments experience.
- `assets/screenshots/` - illustrative reconstructions of key UI screens.
- `assets/mockups/` - design callouts for UX and feature improvements.

## Viewing
Open `CASE_STUDY.md` in your Markdown viewer. Images are linked with relative paths.
